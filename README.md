# Starty

### Base
- TypeScript
- Next.js 15+
- Biome (Linter, Formatter e Analisador de Código)

### Estilização
- TailwindCSS v4
- Radix UI Themes (Componentes pré-construídos)

### Backend & Dados
- Prisma ORM
- Better Auth (Autenticação)
- Resend (Envio de emails)

### Validação
- Zod (Schema validation)
